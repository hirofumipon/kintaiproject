# **KINTAI | 勤怠管理ができるwebアプリケーション**
#### 勤怠管理サービスKINTAIは勤怠管理を簡単にできる管理者用webアプリケーションです。

### URL:[KINTAI](https://kintai-7ly0.onrender.com/)

### サービスへの想い
本業とは関係ない勤怠管理。でも、勤怠管理にかける時間って意外と長いですよね。
そんな雑務を効率化、データ化し、本業に集中できるサービスを作りたい」、そう思い開発をしました。

### 機能
- ログイン機能
- 勤怠データ一元化機能
- 残業アラート
- 勤怠データ追加・修正機能
- 残業時給与自動計算
- 労働基準法参照　など

### 使用技術
##### frontend:HTML/CSS/Bootstrap
##### backend: Python/Django
